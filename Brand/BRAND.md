# CloudPoint brand

CloudPoint's visual idea is **a cloud becoming geometry**. The primary mark is
a luminous point cloud that folds into an abstract `C` and a dimensional
volume. It connects video input, spatial reconstruction, and the product name
without implying photorealistic mesh output.

## Identity

- Product name: **CloudPoint**
- Repository and bundle namespace: `cloud.point.cloud`
- Descriptor: **Local 3D reconstruction for Mac**
- Voice: direct, calm, technically honest, and native to macOS
- Primary typeface: SF Pro / the macOS system font

## Palette

| Token | Hex | Use |
| --- | --- | --- |
| Void | `#07101F` | Main dark surfaces and viewport |
| Cloud Cyan | `#12CFF3` | Primary actions and active points |
| Field Blue | `#287BFF` | Progress and selected state |
| Depth Violet | `#7B4DFF` | Depth accent and secondary glow |
| Ice | `#E8FAFF` | High-contrast text and highlights |

Use cyan and violet sparingly against graphite. Functional status colors remain
the macOS semantic colors so errors, warnings, and success states stay familiar.

## App icon

The 1024-pixel source is `CloudPoint-AppIcon-1024.png`. The Xcode asset catalog
contains the complete 16 through 1024 pixel macOS icon set. Do not add text,
badges, or unrelated AI symbols to the mark.

The master was generated with OpenAI's built-in image generation tool using
this production prompt:

> Create an original premium macOS app icon for CloudPoint: a compact cloud
> made from luminous point-cloud particles and restrained connecting facets,
> folding subtly into a dimensional C and an implied 3D volume. Center it on a
> deep graphite and midnight navy macOS tile. Use electric cyan with sparse
> indigo and violet depth accents. Keep a strong silhouette, generous safe
> margin, and legibility at 16 pixels. Symbol only; no text, watermark, generic
> fluffy cloud, globe, brain, device mockup, or excessive particle clutter.
