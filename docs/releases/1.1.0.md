# CloudPoint 1.1.0

CloudPoint 1.1 adds an experimental Apple SHARP pipeline that turns one selected
video frame or live camera snapshot into a dense metric 3D Gaussian scene,
entirely on Apple Silicon. The app now opens a source first, lets you choose
between the existing multi-frame point cloud and the new Gaussian mode, and
creates the autosaved project only after that choice.

Highlights:

- Apple SHARP inference through PyTorch MPS, with a recoverable CPU fallback.
- A native Metal Gaussian-splat viewer with orbit, pan, zoom, reset, and PLY
  export.
- Seven ranked SHARP keyframes for MOV, MP4, and M4V recordings.
- A live camera preview with one-click snapshot reconstruction.
- Correct upright OpenCV-to-Metal orientation and consistent camera mirroring.
- Strict checkpoint, Gaussian PLY, provenance, and project validation.
- A pinned, self-contained arm64 Python runtime; model weights remain an
  explicit separate download after Apple's research terms are accepted.

Requirements: Apple Silicon and macOS 15.0 or newer. This experimental build is
ad-hoc signed and is not Apple-notarized. Download the ZIP, move `CloudPoint.app`
to Applications, then Control-click the app in Finder and choose Open on first
launch.

SHARP is single-image nearby-view synthesis. It is not multi-view SLAM,
photogrammetry, or a watertight mesh generator.
